Ractive.js SpoilerBox decorator plugin

download: ractive-decorators-spoilerBox.js more plugins at ractivejs.org

SpoilerBox is a box consisting of a header and a content-div implementing the functionality to collapse and expand its content when clicking on its header. Designed as customizable as possible, it can be used in most environments.
Due to various CSS-restrictions, the optional vertical collapse animation was implemented in JavaScript (enabled by passing a duration in ms and an optional easing to the decorator).
Other animations should be straight forward using only css and transitions and the altering (depending on the status of the spoilerBox) css-classes.

The reference Stylesheet can be downloaded from here.
